## Header

Title: Seller views sheet config list
ID: US-015
Priority: Must
Actor: seller
As a: seller
I want: view a list of all sheet configs I have created
So that: I can track my billing configurations and navigate to create bills from them

## Business Value

Sheet configs are the bridge between usage records and bills. A centralized list screen gives sellers visibility into all configured billing setups and serves as the entry point for creating bills.

## Preconditions

- Seller is logged in

## Trigger

- Seller selects "Sheet Configs" from the main menu

## Acceptance Criteria

> Each criterion must be testable and written from the user's or system's observable perspective — no API details.

1. Seller can access the sheet config list screen from the main menu.
2. The list displays all sheet configs created by this seller, sorted by creation date with the most recent first.
3. Each row/card displays: sheet config name and creation date.
4. On desktop, the list is presented as a table.
5. On mobile, each sheet config is presented as a card.
6. The list is paginated.
7. The sheet config name in each row/card is a link. Clicking it navigates the seller to the sheet config detail screen.
8. If no sheet configs exist, the screen displays "No sheet configs found".

## UI/UX References

- None
