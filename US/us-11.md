## Header

Title: Seller creates a usage record
ID: US-011
Priority: Must
Actor: seller
As a: seller
I want: create a new usage record by entering a name
So that: I can collect utility readings from buyers for a billing period

## Business Value

A usage record is the starting point for each billing cycle. Without it, sellers cannot send submission links to buyers or begin the review and billing process.

## Preconditions

- Seller is logged in

## Trigger

- Seller taps "Add Usage Record" on the usage record list screen

## Acceptance Criteria

1. The creation form includes the following field: `name` (required).
2. Submitting the form with `name` empty shows a validation error on that field.
3. Submitting with a name already used by another usage record of this seller shows a validation error.
4. On successful submission, the new usage record appears in the usage record list with status `open`.

## UI/UX References

- None
