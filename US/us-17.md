## Header

Title: Seller views bill list
ID: US-017
Priority: Must
Actor: seller
As a: seller
I want: view a list of all bills I have created across all my buyers
So that: I can track issued bills and navigate to their details for review

## Business Value

The bill list gives sellers a centralized view of all billing activity. Without this screen, sellers have no way to overview what has been issued and to whom, making financial tracking and follow-up impossible.

## Preconditions

- Seller is logged in

## Trigger

- Seller selects "Bills" from the main menu

## Acceptance Criteria

> Each criterion must be testable and written from the user's or system's observable perspective — no API details.

1. Seller can access the bill list screen from the main menu.
2. The list displays all bills created by this seller, sorted by creation date with the most recent first.
3. Each row displays: bill name, buyer name, and creation date.
4. If no bills exist, the screen shows an empty state with a descriptive message.

## UI/UX References

- None
