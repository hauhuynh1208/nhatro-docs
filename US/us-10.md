## Header

Title: Seller views usage record list
ID: US-010
Priority: Must
Actor: seller
As a: seller
I want: view a list of all usage records I have created
So that: I can monitor my usage records, see when each was created, and navigate to their details

## Business Value

Usage records are the entry points for collecting utility readings each billing period. A clear list allows sellers to track which records have been created, their status, and quickly access them for review or approval.

## Preconditions

- Seller is logged in

## Trigger

- Seller selects "Usage Records" from the side menu

## Acceptance Criteria

1. The list displays each usage record with the following information: `name`, `status` (open/closed), `created_at`.
2. On desktop, the list is presented as a table.
3. On mobile, each usage record is presented as a card.
4. The list is paginated.
5. Each row/card has a checkbox (bulk action behaviour to be defined in a separate user story).
6. Each row/card has a "Copy link" button. Tapping it copies the usage record's submission link to the clipboard so the seller can paste and share it via messaging apps.
7. The usage record's `name` in each row/card is a link. Clicking it navigates the seller to the usage record detail screen.
8. If no usage records exist, the screen displays "No usage records found".

## UI/UX References

- None
